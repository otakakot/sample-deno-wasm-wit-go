# sample-deno-wasm-wit-go

## install tinygo

```shell
brew tap tinygo-org/tools
brew install tinygo
```

ref: https://tinygo.org/getting-started/install/

# install wasm-tools

```shell
install.sh
```

```shell
./wasm-tools/wasm-tools --version
```

## install deno

```shell
curl -fsSL https://deno.land/install.sh | sh
```

```shell
npm install @bytecodealliance/jco
```

```shell
deno run index.ts
```
